al momento della creazione di un oggetto, i campi di una classe sono inizializzati automaticamente:

| campo            |     | inizializzato a |
| ---------------- | --- | --------------- |
| int, long        |     | 0, 0L           |
| float, double    |     | 0.0f, 0.0       |
| char             |     | '\0'            |
| boolean          |     | False           |
| classe qualsiasi |     | null            |
